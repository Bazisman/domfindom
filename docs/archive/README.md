# Архив документации

> Обновлено: 2026-04-18

Здесь лежат документы, которые важны как история, но не нужны как основной ежедневный рабочий слой.

## Что сюда перенесено

- старые планы внедрения и миграции;
- промежуточные session-протоколы;
- legacy-документы desktop-эпохи;
- старые checklists и шаблоны, которые уже не являются основным способом работы.

## Как этим пользоваться

- Если нужен текущий контекст — начинать не отсюда, а с `docs/README.md`.
- Если нужно понять, почему когда-то принималось решение, можно поднимать старые материалы отсюда.
- Новые ежедневные записи сюда по умолчанию больше не добавляются.
