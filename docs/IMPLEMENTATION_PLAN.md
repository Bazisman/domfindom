# Implementation Plan

> Создано: 2026-04-05

---

## Правило Ведения Этого Файла

- перед началом этапа или шага он должен быть в статусе `IN PROGRESS`
- после завершения шага он переводится в `DONE`
- если шаг не выполнен, он остаётся `TODO`
- если работа прервалась, по этому файлу должно быть видно, что уже сделано, а что нет

Статусы:
- `[TODO]`
- `[IN PROGRESS]`
- `[DONE]`
