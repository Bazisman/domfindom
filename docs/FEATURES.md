# Возможности продукта

> Обновлено: 2026-04-25

## Аккаунт и безопасность

- регистрация;
- вход и выход;
- подтверждение email;
- восстановление пароля по email;
- удаление аккаунта по email-подтверждению;
- активные сессии;
- журнал событий аккаунта;
- персональное имя профиля (`display_name`);
- темы приложения: `Светлая`, `Темная`, `Система`.

## Личные данные пользователя

- резервная копия данных;
- восстановление данных;
- обнуление данных;
- личные настройки аккаунта;
- выбор режима работы: `Личный` или `Совместный`.

## Финансовые функции

- обзор баланса на главной странице;
- последние транзакции;
- доходы и расходы;
- категории;
- планирование;
- статусы бюджетов по категориям;
- для `daily`-бюджетов статус показывает ожидаемый итог категории на конец месяца, а не полный лимит месяца;
- счета;
- капитал;
- переводы между счетами;
- быстрый ввод по категориям;
- повторяющиеся и плановые операции.

## Семейный режим

- создание семьи;
- приглашение другого пользователя по email;
- встроенные уведомления о приглашениях;
- роли:
  - `Владелец`;
  - `Помощник`;
  - `Только просмотр`;
- семейная сводка;
- семейная история операций на странице `Транзакции`;
- семейный режим в статусе бюджетов, доступный участникам семьи;
- семейные лимиты считаются по лимитам и расходам всех участников, а не по локальным лимитам того, кто открыл экран;
- аудит семейных категорий для владельца;
- предпросмотр и подтверждение связей категорий владельцем семьи;
- показ автора операции по имени профиля;
- доступ к семейной вкладке уже в совместном режиме, даже если семья еще не создана.

## Мобильный интерфейс

- фиксированная нижняя навигация;
- вкладки в одну горизонтальную линию с прокруткой;
- уведомления и аккаунт остаются сверху;
- отключен случайный zoom страницы;
- адаптированные карточки семейного режима.
