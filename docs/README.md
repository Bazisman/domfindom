# 📊 Домашняя бухгалтерия

**Версия:** 1.2.1  
**Дата:** 28 марта 2026

> 🤖 Этот проект поддерживается совместно: человеком и ИИ.

---

## Описание

**Домашняя бухгалтерия** — десктопное приложение для учёта личных финансов.

### Возможности

- ✅ Учёт доходов и расходов
- ✅ Категории транзакций
- ✅ Бюджетирование
- ✅ Управление капиталом (несколько счетов)
- ✅ Автоматические отчисления в капитал
- ✅ Переводы между счетами
- ✅ История операций
- ✅ Экспорт данных в CSV
- ✅ Автоматические бэкапы
- ✅ Логирование

### Технологии

- **Python 3.10+**
- **CustomTkinter** — GUI
- **SQLite** — база данных
- **Asyncio** — асинхронные операции

