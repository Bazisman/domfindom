# Навык: Тестирование изменений

После изменений всегда:

1. Проверка синтаксиса:
`
python -c "import core; import gui; print('OK')"
`

2. Запуск приложения:
`
python run.py
`
Подожди 5 секунд, затем закрой.

3. Проверка логов:
`
Get-Content logs\errors.log -Tail 5
`

Покажи результат проверки пользователю.
